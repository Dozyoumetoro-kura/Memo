## 関数の構造

```
型名 関数名(引数){

処理;

}
```
文字の区切りさえ守ればいいのでこのような書き方もできる(推奨するとは言っていない)
```
int main(){return 0;}
```

# コメントアウト

```
//コメント
```
コンパイラに//の後が無視される。

```
/*コメント*/
```
コンパイラに/*にはさまれた部分が無視される。
デバッグが楽になるので書きましょう。
一時的にプログラムを消したいときも便利。
